1

목적

건축설계 실무자(소장·중간관리자)가 건축 법령을 빠르게 탐색하고, 기준 수치를 즉답 받으며, 복합 법령을 AI로 해석하도록 돕는 도구.

🔍
탐색 속도

25,000+ 조문을 동의어·초성·피인용 순으로 즉시 검색

📐
기준 즉답

용도지역·건물용도·연면적 → 국가 상한 vs 도시 조례 비교, 근거 조문 직접 링크

💬
복합 해석

"적용 가능한지", "단서 조항이 있는지"를 자연어 질의로 확인

대상 도메인

건축법·국토계획법·주차장법·녹색건축물법 등 법령 29개 + 국토부 고시 16개 + 전국 84개 시 조례 309개

그래프 데이터 모델

노드 스키마

// type=article 노드 예시
{
  "id":          "건축법 시행령/제119조",   // <법령명>/<조문번호>
  "type":        "article",
  "law_nm":      "건축법 시행령",
  "article_no":  "119",
  "title":       "면적 등의 산정방법",
  "content":     "<조문 본문>",
  "category":    null,           // null=국가법령, "고시","조례","판례","해석례"
  "domain_tags": ["면적", "높이"],  // 검색·도메인 필터용
  "byeolpyo":    [{ "title": "별표1", "content": "..." }]
}

category: null = 국가 법령(법·시행령·시행규칙). 색상·family 분기에 사용.

엣지 유형

type의미예시
delegates위임건축법 제61조 → 시행령 제86조
cites인용시행령 제119조 → 시행규칙 제43조
applied판례 적용 조문대법원 판결 → 건축법 제14조
interpreted해석례 대상 조문법령해석 20-0419 → 건축법 제2조

피인용 수(citeIn) = applied + interpreted 엣지 수 → 검색 결과 정렬 기준.

패밀리포함 법령

국토계획국토계획법·시행령·시행규칙
주차장주차장법·시행령·시행규칙
분양·녹색건축물분양법·녹색건축물조성지원법 (각 시행령 포함)

기준 카드 데이터 생성

경로 A — 정규식 추출
gen_card_data.py

건폐율·용적률·조경. 100분의 N, N% 등 수치 패턴 추출. 표기 정규화(공백 이름 매칭, "의" 조사 선택, 공장 줄 제외). 소스 미발견 → null.

경로 B — LLM 추출
gen_*_llm.py

주차·이격·심의·완화. graph 별표 원문 → Claude (sonnet-4-6, temp=0) → 표준 JSON. 프롬프트에 9개 표준 용도 사전 주입 → 모델은 수치만 채움.

항목값

  │
  ├── 해시(built_at 제외) 비교 → 변경 없으면 graph.json 원복 후 종료

  │     실패(exit 1) → 파일 미작성, 커밋 안 함

프론트엔드 아키텍처

데이터 로딩 (data.js)

graph.json(~58MB)을 Vite ?url 에셋으로 번들 분리 → 브라우저 런타임 fetch(top-level await). JS 번들 7.3MB gzip → 102KB gzip (98.5% 감소). graph.json은 브라우저 캐시 개별 관리.

심볼용도
nodeByIdid → 노드 O(1) 조회
articlesByLaw법령명 → 조문 배열
citeIn노드 id → 피인용 수 (검색 정렬)
inRel / outRel인용/피인용 엣지 목록
lawColor()category → 색상 팔레트
familyOf()법령명 → 패밀리 레이블
DOMAINS / domainCount도메인 태그 필터용

3가지 UI 모드

모드 1 — 🔍 검색

검색어 파싱: "정확한 구절"(따옴표) · -제외어 · 스페이스 AND. 정렬: 피인용 수 내림차순.

부가: 동의어 그룹 20개 · 초성 검색 · 도메인 칩 필터 · ★ 북마크(localStorage) · URL 딥링크(?q=, ?node=)

모드 2 — 📐 기준 조회

선택 흐름: 지역 스위처(REGIONS) → 서브탭 6종

카드입력출력
ComplianceCard용도지역건폐율·용적률·일조 (국가 vs 도시)
ParkingCard건물 용도부설주차 설치기준 (국가 vs 도시)

데이터 계층: 손큐레이션(*.js, 기존 17개 시) + 자동생성(*_auto.js, 신규 67개 시) → import+spread 병합. 지역 코드: 특·광역시 2자리, 기초시 5자리 법정동코드.

배포 구조

Docker multistage

Stage 1  node:20-alpine
  └── cd web && npm ci && npm run build  →  /dist

Stage 2  python:3.12-slim
  └── pip install -r backend/requirements.txt

Stage 3 (런타임)  python:3.12-slim + nginx :8080
  ├── /dist            Vite 정적 빌드 결과
  ├── data/            graph.json · embeddings.npy

설계 결정 근거 (역추론)

graph.json ?url 분리
